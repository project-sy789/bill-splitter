## ✨ โดดเด่นด้วยฟีเจอร์ระดับพรีเมียม (ใช้งานฟรีทั้งหมด)

| ฟีเจอร์ | รายละเอียด |
|---|---|
| 🗂️ **ระบบบิลครอบจักรวาล** | เพิ่มบิลได้ทั้งแบบจดเอง หรือโยนสลิปให้ระบบอ่าน |
| 📷 **สแกนใบเสร็จอัตโนมัติ (AI-OCR)** | ถ่ายรูปหรืออัปโหลดสลิป ระบบสกัดรายการอาหารให้ทันที รองรับบิลไทย |
| 👥 **จัดการแก๊งเพื่อน (Saved Groups)** | บันทึกกลุ่มเพื่อนประจำ ไม่ต้องพิมพ์ชื่อใหม่ทุกรอบ ดึง PromptPay ติดมาให้ด้วย |
| 💸 **คำนวณเงินฉลาดสุดๆ** | หักลบกลบหนี้ให้อัตโนมัติ ลดจำนวนการโอนเงินไปมาให้เหลือ "น้อยที่สุด" |
| 📊 **ตัวช่วยโอน (Share as Image)** | กด Export รูปสรุปเพื่อส่งเข้าแชท พร้อม QR Code PromptPay ให้เพื่อนสแกนปุ๊บจ่ายปั๊บ |
| ⚡ **ปุ่มรัดเปิด (Quick Tools)** | มีปุ่ม +7% (VAT) และ +10% (Service Charge) คำนวณยอดให้รวดเร็วทันใจ |
| 💾 **ระบบเซฟประวัติ** | มีสมุดประวัติบิล เก็บทุกยอดไว้ในเครื่องโดยอัตโนมัติ |

## 📖 วิธีใช้อย่างรวดเร็ว

### ขั้นตอนที่ 1 — ใส่ชื่อคนหาร
เพิ่มรายชื่อบุคคลที่จะหารบิลด้วยกัน (ใส่เบอร์/เลขบัตรประชาชน PromptPay ไว้เผื่อตอนจบให้เพื่อนสแกนโอนง่ายๆ)
> **Tip:** กด “บันทึกแก๊ง” ไว้ใช้ซ้ำรอบหน้าได้!

### ขั้นตอนที่ 2 — จัดการรายการค่าใช้จ่าย
มี 2 ทางเลือก:
- **กด "+ เพิ่มยอดบิลเอง"** — เมื่อมีแค่ยอดรวมๆ ไม่ต้องการสแกนสลิป
- **กด "ถ่ายหรือเพิ่มรูปสลิป"** — ระบบจะอ่านให้ว่ากินอะไรไปบ้าง ราคาเท่าไหร่

*จากนั้นให้กดเลือกว่า แต่ละไอเทม **"ใครเป็นคนกินบ้าง"** (หารเท่าเฉพาะคนที่เลือก)*

### ขั้นตอนที่ 3 — ใครออกเงินจ่ายไปก่อน?
ระบุยอดเงินที่แต่ละคนเป็นคนสำรองจ่ายให้แม่ค้า เพื่อใช้ในการหักลบหนี้ (หรือถ้าเลือกคนจ่ายจากบิลไปแล้ว ระบบจะรวมยอดมาให้ตรงนี้อัตโนมัติ)

### ผลลัพธ์ — ใครต้องโอนใครบ้าง
ระบบจะคำนวณและสรุปยอดโอนให้ คุณสามารถกด **"แชร์สลิป"** เพื่อแชร์รูปสวยๆ ส่งทวงเงินในไลน์ได้ทันที!

## 🛠️ รันในเครื่องตัวเอง (สำหรับนักพัฒนา)

```bash
git clone https://github.com/project-sy789/bill-splitter.git
cd bill-splitter
npm install
npm run dev
```

เปิด http://localhost:5173/bill-splitter/

## 🧱 Tech Stack

| ส่วน | เทคโนโลยี |
|---|---|
| UI Framework | React 19 + Vite + TypeScript |
| Styling | Tailwind CSS |
| หน้าบ้าน | Lucide Icons, html-to-image (Export PNG) |
| ปัญญาประดิษฐ์ (OCR) | Tesseract.js (tha + eng) วิ่งบนเครื่องผู้ใช้ทั้งหมด |
| สร้าง QR Code | quickchart.io (SVG fallback) |
| ฐานข้อมูล | Browser `localStorage` |
