## 🚀 วิธีใช้

### ขั้นตอนที่ 1 — ใส่ชื่อคนหาร
เพิ่มชื่อทุกคนที่จะหารบิลด้วยกัน ใส่เบอร์ PromptPay ถ้ามี (ไม่บังคับ)

### ขั้นตอนที่ 2 — รายการค่าใช้จ่าย
มี 2 วิธี:
- **กด "เพิ่มรายการ"** — พิมพ์ชื่อและราคาเอง
- **กด "สแกนสลิป"** — ถ่ายรูปหรืออัปโหลดใบเสร็จ ระบบอ่านให้

จากนั้นเลือกว่าแต่ละรายการ **ใครกิน/ใช้บ้าง**

### ขั้นตอนที่ 3 — ระบุใครจ่ายไปแล้ว
ถ้าคนเดียวออกเงินทั้งหมด ไม่ต้องแก้อะไร

### ผลลัพธ์ — ใครโอนให้ใคร
ระบบคำนวณให้อัตโนมัติ กด QR เพื่อโอนผ่าน PromptPay ได้เลย

## 🔬 ระบบสแกนสลิป (OCR)

ใช้ **Tesseract.js** ที่ทำงานบนเครื่องคุณทั้งหมด — ไม่ส่งรูปไปที่ server ใดๆ

มีการปรับภาพก่อนส่งให้ Tesseract อ่าน:
- ขยายภาพเล็กให้ชัดขึ้น
- แปลงเป็นขาวดำ และปรับ contrast
- Adaptive binarization — รองรับแสงไม่สม่ำเสมอ

รองรับ Pattern ใบเสร็จไทยหลายรูปแบบ:
```
ข้าวผัดกุ้ง                120.00   ← แบบมาตรฐาน
01 ข้าวมันไก่               80.00   ← มีเลขนำหน้า
ต้มยำกุ้ง  2x90           180.00   ← จำนวน × ราคา/ชิ้น
```

> **หมายเหตุ:** OCR แม่นประมาณ 60–75% ขึ้นอยู่กับคุณภาพภาพ
> แนะนำให้ถ่ายภาพตรงๆ แสงพอ ไม่เบลอ แล้วตรวจสอบรายการอีกครั้ง

## 🛠️ รันในเครื่องตัวเอง

```bash
git clone https://github.com/project-sy789/bill-splitter.git
cd bill-splitter
npm install
npm run dev
```
